# Heading

```
# Heading1
## Heading2
### Heading3
#### Heading4
##### Heading5
###### Heading6
NO Heading
```

<br>

# 결과

# Heading1

## Heading2

### Heading3

#### Heading4

##### Heading5

###### Heading6

NO Heading

*원래는 Heading1밑에 밑줄이 그어져야하지만 minimal-mistake테마가 Heading2에 밑줄이 그어지게 하는것 같다*

<br>
<br>

# 리스트

```
* 글머리기호1
	* 글머리기호1-1
- 글머리기호2
	- 글머리기호2-1
1. 리스트1
2. 리스트2
```

<br>

# 결과

* 글머리기호1
  * 글머리기호1-1
- 글머리기호2
  - 글머리기호2-1
1. 리스트1
2. 리스트2
